Example applications

The example pages above use page-specific JavaScript code. But if you're developing a more complex web application, you might want to write some cross-page JavaScript files and use them for many web pages. In these pages you then mainly setup the functionality by annotating your HTML with class attributes and load your central JavaScript and CSS files.
Some JavaScript and CSS files for some example applications of the plug-ins are included and get demonstrated by the following demo pages. You may use these script and CSS files directly if they suit your needs, or you may simply see them as examples or blue-prints for your own scripts and styles.

  • enable-default-links
  • accordion (uses the included files accordion(-min).js and accordion.css)
